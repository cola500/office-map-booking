# Interactive Office Map Booking

## Hypothesis

Users will find it faster and more intuitive to book a workspace by clicking directly on a visual office map in SharePoint, compared to selecting a workspace from a traditional list.

## Test

Build a simple web-based prototype with:
- A floor plan image as background
- Clickable hotspots representing workspaces
- Hover highlights and tooltips
- Click-to-book flow opening an Outlook booking link

Embed or demo the prototype within a SharePoint context and gather user feedback.

## Success Criteria

- Users can identify and select a workspace within 5 seconds
- Users prefer the visual approach over a list-based alternative
- The prototype is technically feasible to embed in SharePoint (SPFx compatible)
